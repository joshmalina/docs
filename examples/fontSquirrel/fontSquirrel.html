Demo of GoodDog font from Font Squirrel


Step 1:
Find the font you like on Font Squirrel and download the full font kit (not just the .otf file).

The full kit will include all the file:
  • TTF - Works in most browsers except IE and iPhone.
  • EOT - IE only.
  • WOFF - Compressed, emerging standard.
  • SVG - iPhone/iPad.
Upload all these files to your directory; to keep things clean you should probably create a folder just for them.

Step 2:
Include this snippet of CSS in your styles (replacing the names with your chosen font) 
		@font-face {
			    font-family: 'GoodDogRegular';
			    src: url('GoodDog_font/GoodDog-webfont.eot');
			    src: url('GoodDog_font/GoodDog-webfont.eot?iefix') format('eot'),
			         url('GoodDog_font/GoodDog-webfont.woff') format('woff'),
			         url('GoodDog_font/GoodDog-webfont.ttf') format('truetype'),
			         url('GoodDog_font/GoodDog-webfont.svg#webfontFrKKDeui') format('svg');
			    font-weight: normal;
			    font-style: normal;
			
			}


Step 3:
Assign your new font to an element on your page. For example, we can create a header class and tell it to have the font: 
.header {font: 60px/68px 'GoodDogRegular', Arial, sans-serif; letter-spacing: 0;}
